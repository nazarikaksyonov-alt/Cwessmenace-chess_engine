============================================================
              CWESS MENACE – ШАХМАТНЫЙ ДВИЖОК
              (на битбордах, Python)
============================================================

Это простой шахматный движок с графическим интерфейсом (Tkinter).
Использует битборды для представления фигур, альфа-бета-отсечение
и настраиваемую глубину поиска.

------------------------
БЫСТРЫЙ СТАРТ
------------------------

1. Установите Python версии 3.8 или выше.

   Если Python не установлен, используйте установщик
   "python-manager-26.2.msix"  скачайте с python.org
   ВАЖНО: во время установки обязательно поставьте галочку
   "Add Python to PATH" – это необходимо для корректного запуска.

2. Запустите движок.

   Дважды кликните по файлу "Launch.bat" (или "run.bat").
   Откроется окно консоли, а затем появится графический интерфейс.

   Если движок завис или не отвечает, подождите около минуты –
   возможно, он обдумывает ход (особенно при большой глубине).

3. Смена языка.

   Программа читает настройку языка из файла "lang.txt".
   Чтобы переключиться на русский, напишите в файле:
       ru
   (или "rus"). Для английского – напишите:
       en
   (или "eng"). Если файл отсутствует или содержит что-то другое,
   по умолчанию используется русский язык.

4. Управление.

   - Нажмите на фигуру, чтобы выбрать её, затем нажмите на подсвеченную
     клетку, чтобы сделать ход.
   - Используйте кнопки справа:
       "Новая игра"    – начать заново
       "Сменить цвет"  – играть белыми или чёрными
       "Выход"         – закрыть программу

------------------------
УСТРАНЕНИЕ ПРОБЛЕМ
------------------------

- Если окно не появляется, проверьте, установлен ли Python и
  доступны ли модули (tkinter, copy и др.). Все модули стандартные,
  кроме tkinter (в Linux может потребоваться установить python3-tk).

- Если появляются ошибки о "bit_count", значит ваша версия Python
  слишком старая (нужна 3.8+). Обновите Python.

- Если движок играет слабо, попробуйте увеличить MAX_DEPTH в файле Depth.py
  (но поиск станет медленнее). Для обычной игры рекомендуется глубина 3–4;
  для сильной игры – 5–6 (ход может занимать несколько секунд).

- Движок иногда может делать нелегальные или странные ходы – это
  рабочая версия, возможны ошибки. Сообщайте о найденных багах.

------------------------
СОСТАВ ФАЙЛОВ
------------------------

CwessMenace.py   – главный GUI и локализация
Calculations.py  – доска, генерация ходов, оценка
Depth.py         – константы (глубина, ценности фигур и т.д.)
AlphaSort.py     – сортировка ходов (эвристика истории)
lang.txt         – выбор языка (ru/en)
Launch.bat       – bat-файл для запуска
